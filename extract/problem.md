Mot so luu y cai tien cho extractor:
- Khi extract phai luu lai danh sach cac file da extracted. Vi luong file can extract qua lon va so luong request / day limited, nen khi dat gioi han thi co the tam dung extract va ngay mai co the tiep tuc extract ma khong phai extract lai cac file cu.

- Khi extract phai tinh toan so luong token ma model co the xu ly. Vi du prompt + file.md de extract co luong token vuot qua luong token ma model co the xu ly thi can phai chia chunk hoac su dung mot so ki thuat nao do de toi uu input cho model extract. 

- Boi vi toi su dung free model nen co the can nhieu provider hoac nhieu api-key tu cac providers do. Nen can phai xu ly code sao cho co the chay muot ma voi toan bo key ma toi paste vao.
